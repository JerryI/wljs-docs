## How to find main axes of a sampled ellipse
We need follow the steps

```mathematica
(* Sample data points representing an ellipse *)
data = makeCurve[test, 550];

(* 1. Center the data *)
centeredData = With[{m = Mean[data]}, (# - m )&/@ data ];

(* 2. Compute the covariance matrix *)
covMatrix = Covariance[centeredData];

(* 3. Find the eigenvalues and eigenvectors of the covariance matrix *)
{eigenvalues, eigenvectors} = Eigensystem[covMatrix];

(* The eigenvectors correspond to the main axes of the ellipse *)
mainAxes = eigenvectors

(* The eigenvalues correspond to the lengths of the axes *)
axesLengths = eigenvalues

ListLinePlot[data, Epilog->{
  Line[{{0,0}, #}] &/@ mainAxes
}, AspectRatio->1]
```
