WLJS

Open-source Wolfram Frontend & Interpreter & Fullstack Framework written in Javascript

Getting started - 5min ⏱️

Math input

Mix math and code together using oustanding CodeMirror 6 editor.

Customized cells

Write in Markdown, Javascript, HTML, Mermaid or add a language you like.

GUI Building blocks

Most essential UI elements are provided (sliders, text-boxes, buttons).

SVG Graphics

Most Wolfram Mathematica's plotting functions were recreated using modern web-tools like D3, Plotly & THREE.

Syntax sugar

Make your own custom representation of any defined symbol using HTML, CSS, JS or just WL

Embed & Export. Run locally or in a cloud

Share your notebook as a standalone .html file or React component keeping graphs interactive.
