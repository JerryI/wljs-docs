## Interface
Pair your JoyCon with your PC (unpair from Switch console firsts), and then click to connect on the single button

![](./../../../Screenshot%202025-03-18%20at%2012.45.05.png)

Then you regular interface for bindings [`EventHandler`](frontend/Reference/Misc/Events.md#`EventHandler`)

```mathematica
EventHandler["JoyCon", {
  "zr" -> (FrontSubmit[FrontSlidesSelected["navigateNext", 1]]&),
  "y" -> (FrontSubmit[FrontSlidesSelected["navigatePrev", 1]]&),
  "Stick" -> Print (* for example *)
}];
```
