Used to replace wrapped Wolfram Expression with custom HTML elements

```mathematica
FrontEndView[expr_, "1:compressed string"]
```

Most decorations generated by `TemplateBox` and etc...

- the expression is __not editable__
- a common WL editor __is now spawned__ inside
- the inner `expr` __is revealed__ after first the evaluation 

`RGBColor`, `DateObject` uses this structure to display a human-readable form of the content inside

If you are looking for something to decorate the expression keeping it editable, use [FrontEndBox](../../Decorations/Internals/FrontEndBox.md) or [FrontEndBoxTemporal](../../Decorations/Internals/FrontEndBoxTemporal.md)
