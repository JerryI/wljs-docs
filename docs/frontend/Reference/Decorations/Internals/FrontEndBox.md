Used to decorate wrapped Wolfram Expression with custom HTML elements

```mathematica
FrontEndBox[expr_, "1:compressed string"]
```

Most decorations generated by `Frame` and etc...

- the expression is __editable__
- a common WL editor is spawned inside
- only the container of the editor can be decorated
