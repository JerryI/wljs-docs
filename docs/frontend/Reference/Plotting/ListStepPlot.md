```mathematica
ListStepPlot[{y1, y2, ...}]
```
plots the values $y_1$, $y_2$ ... in steps 
## Examples
Using $x$ and $y$ coordinates
```mathematica
ListStepPlot[Table[{Prime[n], EulerPhi[n]}, {n, 50}]]
```
