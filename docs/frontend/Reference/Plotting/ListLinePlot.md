Plots a line like [Plot](Plot.md) through the list of points provided
```mathematica
ListLinePlot[{y1, y2, y3, ...}]
```

## Options
The same as for [Plot](Plot.md)

## Examples

```mathematica
ListLinePlot[{1, 1, 2, 3, 5, 8}]
```
